# Link 外链组件

该组件暴露了允许在 app 内唤起其他应用的方法。如发送短信、打电话、发邮件、打开外链、打开系统设置。

## API

### Link.email(email: string)

### Link.settings()

### Link.sms(phoneNumber: string)

### Link.call(phoneNumber: string)

### Link.url(url: string)

# List 列表组件

## 效果演示

## API

| 属性                | 必填    | 说明                 | 类型              | 默认值 |
| ------------------- | ------- | -------------------- | ----------------- | ------ |
| title               | `true`  | 标题                 | `ReactNode`       |        |
| items               | `true`  | 列表项               | `ListItemProps[]` |        |
| itemBackgroundColor | `false` | 统一设置列表项背景色 | `主题颜色`        |        |
